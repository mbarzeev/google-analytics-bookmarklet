To translate Capriza user id to email address in Google Analytics, Drag the following link to your bookmarks bar:

Transform User Ids

Just hit the bookmark once after the page is loaded, while already logged in to Capriza manage. The script will continuously monitor the website for user id and translates it to email address based on Capriza database

We use this script since Google Analytics ToS does not allow storing personal emails in their database
